## Projeto loja virtual

Este projeto faz parte de minha aprendizagem sobre JDBC, pela Alura.

Assuntos abordados:
- Introdução o JDBC e o padrão Factory;
- Executando comando SQL no Java;
- Evitando o SQL injection;
- Controle de transação;
- Camada de persistência com o DAO;
- Evitando queries N+1;
